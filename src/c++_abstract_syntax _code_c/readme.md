## 项目介绍
此项目主要是分析产生式 按照自顶向下的分析方法
1. 求出first集合follow集 
2. 最终得到select集  
3. 构造ll1分析表
4. 输入一个串判断该串是否是该文法的语言


这个项目主要涉及到
1. productionForm.h    // 产生式类头文件
2. productionForm.cpp   //产生式cpp文件
3. LL1.h   // LL1类的头文件
4. LL1.cpp
5. main.cpp // 主执行文件  其实就是构造了一个类  
    构造函数传入读入的文件路径
   
文件input的内容
1. 第一行 所有的终结符
2. 第二行 所有的非终结符
3. 第3行到第3+n行 n是文法的产生式数量 
4. 第3+n+1行 输入要匹配的串  程序将判断是否符合语言的文法 

example文件中记录了可行的范例
